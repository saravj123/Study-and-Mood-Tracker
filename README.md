# Study-and-Mood-Tracker

## Overview
This project is an app that lets you track how many hours you study for and how you feel each day.

## Features
- Study hours tracked
- Daily moods tracked (moods are color coded as well)
- Reset button to clear tracker

## How to Use
1. Enter the number of hours studied
2. Select your mood
3. Click "Save Entry"
4. View your entries below

## Languages
- HTML
- CSS
- JavaScript
